# Playwright Login Automation Project

## Description
This project demonstrates end-to-end (E2E) test automation using **Playwright**.
It includes a simple login web application and automated tests written in JavaScript,
following **Page Object Model (POM)** best practices.

The goal of this project is to showcase practical Playwright skills suitable for a
Junior QA Automation role.

---

## Tech Stack
- JavaScript
- Playwright
- HTML / Vanilla JS
- Node.js
